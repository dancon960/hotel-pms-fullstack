# Hotel PMS Fullstack

### Descripción
Este proyecto es un sistema de gestión hotelera (Property Management System) diseñado para centralizar la operativa de Front Office. Permite el control de disponibilidad de habitaciones en tiempo real, la gestión de reservas y la coordinación del estado de limpieza entre departamentos.

### Tecnologías Principales
* **Frontend:** React + TypeScript + Tailwind CSS
* **Backend:** Node.js + Express
* **Gestión de Proyecto:** Metodología Agile (Kanban)
